```text
Estado 0 = 100% Texto

```

Isso significa que você não criou especialização emergente.

```text
Tarefa Texto
↓
Estado 0 sempre ganha

Tarefa Código
↓
Estado 1 sempre ganha

Tarefa Matemática
↓
Estado 2 sempre ganha

Tarefa Planejamento
↓
Estado 3 sempre ganha
```

Ou seja:

```text
Categoria → Estado Fixo
```

Não existe competição real.

```python
estados[0].afinidade['texto'] = 2.0
estados[1].afinidade['codigo'] = 2.0
```

Depois faça reforço apenas quando ganhar:

```python
vencedor.afinidade[categoria] += 0.01
```

Mas também faça esquecimento:

```python
for cat in categorias:
```

```text
vira especialista
```

sem você dizer que ele deveria ser.

# O problema do Estado 4

Estado 4 nunca ganhou.

Isso é um sinal importante.

Na sua arquitetura MultiEstado ele seria equivalente a:

```text
Neurônio morto
```

ou

```text
Especialista que nunca recebe tarefas
```

Você precisa introduzir exploração.

Exemplo:

```python
if random.random() < 0.05:
    vencedor = random.choice(estados)
else:
    vencedor = max(...)
```

5% das vezes um estado aleatório recebe a tarefa.

Isso impede neurônios mortos.
