# V4 - Arquitetura MultiEstado

## Objetivo

Construir uma arquitetura de agentes baseada em neurônios MultiEstado organizados em uma hierarquia:

```text
CEO
 ↓
Manager
 ↓
Worker
```

Cada agente possui sua própria camada V4 composta por múltiplos neurônios MultiEstado.

---

## Estrutura Atual

```text
core/
├── neuron_multistate.py
└── v4_layer.py

agents/
├── ceo_layer.py
├── manager_layer.py
└── worker_layer.py

runtime/
└── pipeline.py
```

---

## MultiStateNeuron

Cada neurônio possui:

* múltiplos estados
* memória por estado
* roteamento baseado na entrada

Exemplo:

```text
Estado 0
Estado 1
Estado 2
```

Uma entrada ativa apenas um estado por vez.

---

## V4Layer

A camada V4 contém vários neurônios.

Exemplo:

```text
V4Layer
├── Neuron 0
├── Neuron 1
├── Neuron 2
└── Neuron 3
```

Cada neurônio recebe uma versão da entrada e produz seu próprio estado.

---

## Hierarquia Atual

### CEO

```python
V4Layer(num_neurons=4, states=3)
```

Responsável pelo planejamento inicial.

---

### Manager

```python
V4Layer(num_neurons=6, states=4)
```

Responsável pelo refinamento.

---

### Worker

```python
V4Layer(num_neurons=8, states=5)
```

Responsável pela execução.
