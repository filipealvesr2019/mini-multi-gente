# 🔹 Conclusão científica

1. **Especialização emergente funciona parcialmente**

   * Estados conseguem se especializar sem boost artificial.
2. **Distribuição de carga ainda é instável**

   * Colapso parcial ocorre em seeds diferentes → a seed 123 é um exemplo claro.
3. **Sinais para próximos passos**

   * Precisamos de um mecanismo mais robusto de **balanceamento adaptativo** que previna estados dominantes de saturar a simulação.
   * Possíveis soluções:

     * Penalidade não linear para estados muito ativos.
     * Softmax com temperatura adaptativa (reduz probabilidade de saturação).
     * Interação multiestado (rede de neurônios) ao invés de um único MultiEstado.
